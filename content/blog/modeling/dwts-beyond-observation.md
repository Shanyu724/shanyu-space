## 问题：不可观测的观众投票

《与星共舞》（DWTS）的淘汰机制融合了两条通道：评委专业打分和观众投票。但观众投票数据本身不可观测——我们能看到的只有评委打分、淘汰结果和赛季排名，而决定胜负的关键变量是黑箱。

更复杂的是，34个赛季中投票聚合规则在不断变化。早期使用Rank Rule（排名加权），后期切换为Percent Rule（百分比加权），第28季后又引入了评委救助机制（Judge Save）。规则变异让跨赛季的比较变得困难，也让争议不断——Bobby Bones（第27季冠军）在评委打分持续垫底的情况下依靠人气夺冠，引发了关于"技术 vs 人气"的广泛讨论。

2628171号论文的核心问题可以概括为：给定不完整的观测数据——评委打分、淘汰序列、选手属性——能否反演出不可见的观众支持率，并在此基础上比较不同规则的系统性偏差、诊断偏差的来源、设计更稳健的投票机制？

这是一个典型的反问题（inverse problem）：已知系统的输出，推断隐藏的状态和参数。

## 数据预处理：从原始数据到结构化面板

在建模之前，论文完成了一套系统的数据预处理流水线。原始数据来自 `2026_MCM_Problem_C_Data.csv`，初始为“选手-赛季”宽表格式。

预处理做了三件事：

**纵向面板重构。** 将宽表转化为“选手-周次-赛季”长面板，得到2777条“选手-周”记录，覆盖34个赛季、421名选手。逐周定义活跃选手集 `A_{s,t} = {i: J_{ist} > 0}`，只保留当周有有效打分的选手，剔除淘汰后的观察值，避免生存偏差。

**标准化与正态化。** 评委打分在不同赛季之间存在尺度差异，需要消去这些系统差异。先将原始打分转化为周内份额：

`q_{ist} = J_{ist} / Σ_{k∈A} J_{kst}`，`q ∈ (0,1)`

然后对份额做Logit变换，映射到实数域：

`y^(J)_{ist} = ln(q_{ist} / (1 - q_{ist}))`

观众投票份额 `v_{ist}` 同样做Logit变换。核密度估计验证变换后的分布近似正态，满足后续线性混合效应模型的同方差假设。

**缺失值处理。** 评委缺失或特邀评委等结构性缺失，采用动态求和而非均值插补，保留原始方差信息。类别型协变量缺失编码为 `Unknown`，保留完整样本量。

## 四个假设

论文提出了四个基本假设，每个都有明确理由：

**假设1——人气具有马尔可夫性。** 选手的隐性人气在周与周之间连续演化，当周人气仅依赖于前一周的状态和当周的随机扰动。排除了“断崖式突变”的可能性，除非有外生冲击。

**假设2——评委打分独立于投票规则。** 评委打分发生在聚合规则应用之前，因此反事实回放时可以固定历史评委数据不变，只改变聚合方式。

**假设3——淘汰是概率事件。** 不是严格的排序决定论——评委救助等机制引入主观随机性，因此淘汰概率高不等于必然发生。

**假设4——表现可分解为加性组合。** 选手表现 = 选手特征的固定效应 + 舞伴的随机效应 + 时间趋势。线性结构在有限样本下提供了最稳健的解释力。

## Task 1：隐变量反演——HB-SSM

### 为什么需要状态空间模型

直接反推投票份额面临三重困难：信号截断与稀疏性（淘汰只提供序数约束而非连续反馈）、马尔可夫动力学（人气跨期相关）、规则异质性与主观噪声（不同规则+评委救助）。状态空间模型天然适合这类问题——它同时建模隐状态的演化和观测数据的生成过程。

### 状态方程

引入潜在对数支持度 `x_{i,s,t}`，假定它服从带漂移项的随机游走：

`x_{i,s,t} = x_{i,s,t-1} + δ_{i,s} + ε_{i,s,t}`，`ε ~ N(0, σ²_s)`

其中 `δ_{i,s}` 是选手 `i` 在第 `s` 赛季的漂移项，`σ²_s` 是赛季 `s` 的总体人气波动方差。漂移项的存在意味着模型允许选手人气有长期趋势。

### Softmax变换

将连续隐状态映射到满足概率单纯形约束的投票份额：

`v_{i,s,t} = exp(x_{i,s,t}) / Σ_{k∈A} exp(x_{k,s,t})`

这个变换保证了每期所有选手的投票份额之和为1，且份额在0到1之间。更重要的是，Softmax的指数形式放大了隐状态之间的差异。

### Plackett-Luce排序似然

淘汰不是确定事件，而是概率事件。给定风险指数 `b_{i,s,t}`，选手被淘汰的概率为：

`P(E=e | b) = exp(λ · b_e) / Σ_{k∈A} exp(λ · b_k)`

`λ` 是规则刚性因子。`λ→∞` 时淘汰完全由规则排序决定；`λ→0` 时淘汰接近均匀分布。论文最终取 `λ = 5.0`。

两种规则下的风险指数定义不同：

- `Rank Rule`：`b_i = Rank(J_i) + Rank(v_i)`
- `Percent Rule`：`b_i = -(q_i + v_i)`

### 求解方法

采用MCMC采样（Metropolis-Hastings算法），在全参数空间上探索后验分布。最优配置为 `λ = 5.0`，`σ_s = 0.5`。

### 结果

一致性检验覆盖全部264次淘汰事件：

- 总体 Perfect Consistency Rate：55.3%
- Percent Rule 子集：73.7%
- Rank Rule 子集：3.0%

Percent Rule 的73.7%匹配率说明该规则在执行上高度刚性；而 Rank Rule 的3.0%匹配率则暗示该时期存在大量规则外干预。

## Task 2：反事实推演——Counterfactual Replay与决策优化

### 后验采样回放系统

有了Task 1生成的K组后验样本 `{v^(k)}`（`K=500`），就可以做反事实实验。固定历史评委打分 `J` 和选手阵容不变，只改变聚合算子 `F(·)`，观察淘汰结果会如何变化。

核心指标是因果翻转概率 `P_diff`——规则切换导致 Bottom-2 选手改变的概率：

`P_diff(s,t) = (1/K) Σ I(E^(k)_Rank ≠ E^(k)_Percent)`

### Logit评委救助模型

评委在 Bottom-2 中决定拯救谁，是一个离散选择行为。用Logit模型拟合：

`P(Save i | {i,j}) = 1 / (1 + exp(-β^T Δx))`

拟合结果显示，技术打分份额的系数显著为正，说明评委救助明显偏向技术分更高的选手。

### 规则敏感性

在20个关键周的采样回放中，`P_diff = 0.151`——约15%的淘汰结果会因规则改变而不同。

信号偏差量化中，Percent Rule 与原始观众排名的相关性为0.917，Rank Rule 为0.767。Percent Rule 更忠实地传递了观众意愿。

### Minimax Regret决策推荐

论文用 Minimax Regret 准则比较了四种方案：

- Rank Rule：0.1676
- Percent Rule：0.0777
- Rank + Save：0.1098
- Percent + Save：0.0190

结论是：`Percent + Save` 在偏好不确定下的最大后悔最小。

## Task 3：双通道归因——Dual-Channel Mixed-Effects Model

### 为什么要双通道

单通道模型无法区分“因为跳得好”还是“因为受欢迎”。论文把输出拆成两个平行响应变量：评委打分份额的Logit变换 `y^(J)` 和观众投票份额的Logit变换 `y^(F)`。

### 模型公式

对于通道 `c ∈ {J, F}`，拟合线性混合效应模型：

`y^(c)_{ist} = α^(c) + x_i^T β^(c) + δ_s + κ_t + b^(c)_{p(i)} + u^(c)_{p(i)} τ_t + ε^(c)_{ist}`

这个结构同时吸收了赛季效应、周次效应、舞伴效应和选手特征效应。

### 结果

**年龄效应的不对称性。** 年龄在观众通道的负向影响显著强于评委通道，说明观众比评委更容易受选手本身特征影响。

**舞伴的双重增强效应。** 舞伴在技术与人气两个通道上的能力高度正相关，说明顶级舞伴不仅能提升技术表现，也能提升观众支持。

**稳健性。** Leave-One-Season-Out 验证表明这些结论具有跨赛季稳定性。

## Task 4：机制设计——Dispersion-Matching Nonlinear Compression

### 问题的结构性诊断

Percent Rule 虽然整体优于 Rank Rule，但观众投票的方差往往远大于评委打分的方差。简单线性加权会让观众投票的长尾分布淹没技术信号，这就是方差稀释效应。

### 非线性压缩算子

引入压缩指数 `γ ∈ (0,1]`，对观众投票做幂律变换后重新归一化：

`ṽ_{ist}(γ) = v^γ_{ist} / Σ_{k∈A} v^γ_{kst}`

当 `γ < 1` 时，极端高票的边际权重被压缩，但排名顺序不改变。

### 自适应参数求解

优化目标是让观众投票的离散度与评委打分离散度对齐。如果观众离散度本身不超过评委，则 `γ = 1`，不做干预；否则用 Brent 方法求解最优 `γ`。

### 最终聚合规则

综合得分：

`C_{ist} = q_{ist} + ṽ_{ist}(γ_st)`

评委份额与压缩后的观众份额等权相加，`γ_st` 每期自适应求解。

### 效果统计

对335个赛季-周样本模拟显示：

- 触发率：88.4%
- 平均压缩强度：0.4175
- 压缩前离散度：0.0172
- 压缩后离散度：0.0014
- 82.7%的淘汰结果保持不变

这说明新机制是“最小干预”的：只在方差失调时介入。

## 备忘录

论文最终给制作方三条建议：

1. 采用 `Percent聚合 + Bottom-2评委救助`
2. 提高透明度，公开评委与观众分项占比
3. 试点自适应压缩机制

## 方法论反思

这篇论文最有价值的，不只是模型精度，而是它把“不可观测变量反演”“反事实推演”“归因拆解”和“机制设计”连成了一条完整链路。

它不是单纯在预测结果，而是在回答一个更有力量的问题：当规则改变时，系统会如何改变；而当系统暴露出结构性偏差时，我们能否设计出更稳健的机制去修正它。

## 模型的核心逻辑

IS-LM 模型是宏观经济学中连接**实体经济**与**货币市场**的核心分析框架。

- **IS 曲线**（投资-储蓄）：描述产品市场均衡——投资等于储蓄时，利率与产出的关系
- **LM 曲线**（流动性偏好-货币供给）：描述货币市场均衡——货币需求等于货币供给时，利率与产出的关系

两条曲线的交点决定了短期内的均衡利率和均衡产出。

## IS 曲线的推导

IS 曲线的逻辑链条：

```
利率下降 → 投资增加 → 总需求增加 → 产出增加
```

从代数角度来看，在两部门经济中：

```
Y = C + I
C = a + bY
I = e - dr
```

解得：Y = (a + e - dr) / (1 - b)

因此 IS 曲线斜率为负，利率 r 与产出 Y 负相关。

## LM 曲线的推导

LM 曲线的逻辑链条：

```
产出增加 → 交易性货币需求增加 → 利率上升（给定货币供给不变）
```

货币需求函数：L = L₁(Y) + L₂(r)，其中 L₁ 是交易需求（与 Y 正相关），L₂ 是投机需求（与 r 负相关）。

因此 LM 曲线斜率为正，产出 Y 与利率 r 正相关。

## 均衡与政策含义

### 财政政策

扩张性财政政策（增加 G 或减税）使 IS 曲线右移，导致利率和产出同时上升。但利率上升会**挤出**部分私人投资。

### 货币政策

扩张性货币政策（增加货币供给）使 LM 曲线右移，降低利率、增加产出。

## 关键辨析

- **流动性陷阱**：当利率极低时，LM 曲线水平，货币政策失效，财政政策效果最佳
- **古典情形**：LM 曲线垂直时，财政政策完全挤出，货币政策效果最佳

这个模型虽然简化，但提供了理解宏观政策传导机制的基础框架。后续学习中，IS-LM-BP 模型将进一步把开放经济引入分析。

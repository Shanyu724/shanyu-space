# 番茄钟（Pomodoro）

一个**安静的番茄工作法计时器**。

## 它是什么

- 一个跑在网页上的计时器，三种模式：Focus（25 分钟）· Short Break（5 分钟）· Long Break（15 分钟）
- 每完成 4 轮 Focus，自动进入 Long Break
- 所有设置（时长、自动连跑、提示音）存浏览器本地，关页面回来不丢
- 累计的 Focus 次数也会持久化，可以看自己一周/一天做了几轮

## 怎么用

- **空格键**：开始 / 暂停
- **R 键**：重置当前轮
- 顶部三个按钮切换模式
- 右上角齿轮进入设置：调整时长、长休间隔、是否自动跑下一轮、是否开声音

## 适合什么场景

- 写长文、做作业、读论文这种需要连续专注 25 分钟+ 的任务
- 不适合频繁被打断的工作（这时反而应该开 Short Break 模式）
- 也可以反过来用——设 5 分钟 Focus 模式当 sprint 节奏

## 技术细节（用户问"怎么做的"才说）

- Next.js 15 Client Component，纯前端，无后端
- 用 `setInterval` 驱动计时，避免 `setTimeout` 漂移
- 倒计时用 `Math.max(0, ...)` 防止负数
- 铃声用 Web Audio API 合成「叮咚」（两个正弦波 + ADSR 包络），零外部音频依赖
- 进度环是 SVG + `stroke-dasharray` / `stroke-dashoffset` 实现
- 状态全部 localStorage，键名带 `pomodoro:` 前缀避免冲突
- 动画用 framer-motion（与站点一致）

## 设计取向

- 不做成就系统、不做任务清单、不做数据可视化大盘
- 安静、专注在"一个数字在倒数"这件事上
- 离开键盘用空格、回到键盘也用空格——保持节奏不被打断
